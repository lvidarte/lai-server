Lai

Welcome to lai, a program to manage and sync linux commands, code snippets or any anotation that developers or sys admin need get quickly in theirs console terminals.

Download the lai client.

Client

Add some text

$ lai --add 'grep -R <pattern> --include \*.txt'

Search some text

$ lai grep
1: grep -R <pattern> --include \*.txt

Make public the doc

$ lai --set-public 1

Sync with lai-server

$ lai --sync

Search in public docs

$ lai --server-search grep
5015d7273042976dc5000230: egrep -w 'word1|word2' /path/to/file
5015d7363042976dc500031b: grep -i ps ~/.bash* | grep -v history

Copy a doc

$ lai --copy 5015d7273042976dc5000230

Server

You can download and set your own lai server or use the public lai server.

Whatever the chosen server, just login with your Google account and add a public key with a name to identify it. When configure the client use your email and the pub key name to authenticate against the server.

lai is free software.
